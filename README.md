# FreeCodeCamp Python Practice

This repository contains Python exercises completed as part of FreeCodeCamp's curriculum and certificate programs. The focus is on **practical learning**, applying Python concepts, and developing problem-solving skills through small projects and exercises.

---

## About This Repository

- Exercises were completed while following FreeCodeCamp's **Python curriculum**.
- Topics covered include:
  - Data types and structures (`list`, `dict`, `str`, `int`)
  - Input validation
  - Loops and conditionals
  - Functions and modular design
  - Basic file handling and error checking
  - Using regular expressions for data validation
- Projects demonstrate **learning in action**, and reflect my early journey in Python programming.
