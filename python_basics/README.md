# Python Basics Module

Core beginner Python exercises focused on iteration, numeric methods,
and foundational algorithmic thinking.

## Topics Covered
- Loops and conditionals
- Numeric algorithms
- String processing
- Iterative problem solving

## Exercises

| File | Concept | Difficulty |
|------|----------|------------|
| fibonacci_iterative.py | Iterative Fibonacci | Easy |
| square_root_bisection.py | Bisection method | Easy |
| pin_extractor.py | String parsing | Easy |
| generate_parentheses.py | BFS combinations | Medium |

Each script runs independently.
